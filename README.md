# Sistema de Carnet de Empleados

Sistema web desarrollado para la gestión de empleados y generación de carnets institucionales.

## Características

- Autenticación de usuarios
- Búsqueda de empleados por DNI
- Generación de carnet de empleado
- Generación de ficha personal imprimible
- Interfaz web simple y rápida

## Tecnologías utilizadas

- PHP
- MySQL
- Docker
- HTML / CSS
- Docker Compose
